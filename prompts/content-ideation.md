# content-ideation

## 使用目的

用于评估内容选题是否值得做，并优化标题、结构与传播角度。

## 适用输入

适合输入以下信息：

- 内容主题
- 目标平台：公众号 / 小红书 / 抖音 / YouTube / X 等
- 目标结果：传播 / 转化 / 教育 / 品牌
- 当前已有草稿、标题或核心观点
- 你担心的问题：太空、太硬、太不爆、太不清楚等

## 调用哪些 skill

默认调用：MrBeast + Feynman + Zhang Xuefeng  
品牌导向时可补：Steve Jobs

岗位分工：
- MrBeast：传播张力、钩子、标题与结构注意力设计
- Feynman：清晰度与认知价值
- Zhang Xuefeng：现实感、口语化、接地气表达
- Jobs（可选）：统一主张与品牌表达

## 执行步骤

1. 先把问题归类为 `content`。
2. 明确内容目标、平台、受众与希望触发的动作。
3. 让 MrBeast 判断选题张力、开头钩子和传播潜力。
4. 让 Feynman 判断内容是否真的讲清楚了。
5. 让 Zhang Xuefeng 判断表达是否接地气、是否离用户太远。
6. 需要品牌感时加入 Jobs 看主张是否够聚焦。
7. 汇总为统一内容建议：做不做、怎么切、怎么写。
