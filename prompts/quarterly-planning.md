# quarterly-planning

## 使用目的

用于制定未来一个季度的现实可执行计划，而不是只列愿望清单。

## 适用输入

适合输入以下信息：

- 未来 90 天的目标
- 当前资源：时间、现金流、技能、人脉
- 当前主要压力：收入、增长、健康、学习、作品等
- 不可妥协项和可放弃项
- 你目前最纠结的取舍

## 调用哪些 skill

默认调用：Naval + Munger + Taleb  
现实补位：Zhang Xuefeng

岗位分工：
- Naval：长期方向、杠杆、幸福感与可持续性
- Munger：机会成本、配置错误、反向约束
- Taleb：脆弱性、下行保护、留冗余
- Zhang Xuefeng（可选）：现实落地与执行阻力校准

## 执行步骤

1. 先把问题归类为 `planning`。
2. 写清楚未来季度目标、资源、约束和关键选择。
3. 让 Naval 判断哪些方向值得长期投入。
4. 让 Munger 判断哪些事情最该砍、最该避免。
5. 让 Taleb 判断计划是否过满、过脆弱、缺缓冲。
6. 如计划明显理想化，再让 Zhang Xuefeng 做现实校准。
7. 汇总成统一季度计划：主目标、停止事项、关键指标、节奏安排。
